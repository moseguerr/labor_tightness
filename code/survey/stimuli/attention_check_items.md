# Attention Check Items — Study 2

---

## Attention Check 1

**Placement:** After all five ranking tasks, before wages are revealed.

**Question:**

> Which posting explicitly mentioned a mentorship program and an annual learning budget?

**Options (randomized):**

| Option | Text | Correct? |
|--------|------|----------|
| A | Vantage Strategy Group | No |
| B | Bridgewell Financial | No |
| C | Crestline Industries | **Yes** |
| D | Novalink Group | No |
| E | Thornbury Corp | No |

**Rationale:** Posting C (Crestline Industries) explicitly states "a structured mentorship program" and "a dedicated professional development budget of $2,000 per year." No other posting mentions either a mentorship program or a learning budget. Tests whether participants actually read the postings rather than skimming.

---

## Attention Check 2

**Placement:** After the card sort task, before the debrief.

**Question:**

> Which of the following phrases appeared in one of the postings you read?

**Options (randomized):**

| Option | Text | Correct? | Source |
|--------|------|----------|--------|
| A | "reducing inequality" | **Yes** | Posting B (Bridgewell Financial) |
| B | "industry-leading compensation" | No | Plausible but absent — wage cue |
| C | "award-winning workplace culture" | No | Plausible but absent — good-employer |
| D | "accelerating sustainable innovation" | No | Plausible but absent — purpose/innovation |

**Rationale:** "Reducing inequality" appears verbatim in Posting B's mission section. The three distractors are plausible phrases that sound like they could appear in purpose or good-employer postings but do not appear in any of the five stimuli. Tests recognition memory for the actual posting content.

---

## Exclusion Rules

- **Fail both checks:** Exclude from analysis. Record but do not use data.
- **Fail one check:** Flag for sensitivity analysis. Include in primary analysis but run robustness checks excluding these participants.
- **Pass both checks:** Full inclusion.

---

## Implementation Notes

- Check 1 options should be presented as company names, not posting labels (A–E).
- Check 2 options should be presented as quoted phrases.
- Record the selected answer, response time, and whether the answer is correct.
- Do not provide feedback to the participant on whether their answer was correct.
