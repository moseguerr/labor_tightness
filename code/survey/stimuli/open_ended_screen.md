# Open-Ended Post-Ranking Screen — Study 2, Stage 1

Presented after all five ranking dimensions are complete and before wages are revealed.

---

## Instruction Text

> In a few words, what stood out most to you about each posting? You can note anything — the role, the company, the language used, or anything else.

---

## Display Format

Five text input boxes, each labeled with the company name and job title. Maximum 100 characters per box.

| Label | Posting |
|-------|---------|
| **Vantage Strategy Group** — Strategy Analyst | Posting A |
| **Bridgewell Financial** — Financial Analyst | Posting B |
| **Crestline Industries** — Marketing Coordinator | Posting C |
| **Novalink Group** — People Analytics Analyst | Posting D |
| **Thornbury Corp** — Operations Analyst | Posting E |

---

## Implementation Notes

- All five text boxes should be visible simultaneously (no scrolling if possible).
- Display order of the five boxes should match the participant's randomized posting order from the ranking task, to maintain continuity.
- All fields are optional — participants may leave any box blank.
- Character counter should be visible (e.g., "42 / 100").
- Record response text and timestamp per box.
- This item is not scored. It provides qualitative signal-detection data and helps identify failed manipulations in pilot testing.
